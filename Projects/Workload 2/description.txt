Routes
1. get - /
2. get - /listings - to show all the tasks
3. get - /listings/:id - to view all the info about the tasks
4. get - /listings/new - to create a new tasks
5. post - /listings - to post the newly created listings
6. get - /listings/:id/edit - to edit
7. put - /listings/:id - to update the listing in the database
