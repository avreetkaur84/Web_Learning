#HTML LINKS
1. <a href="url">link text</a>
2. Types of link: 1. Unvisited Link
			2. Visited Link
			3. Active Link
3. target: specifies where to open the linked document
	1. _self - Opens the document in same window/tab as it was clicked
	2. _blank - Opens the document in a new window or tab
	3. _parent - Opens the document in the parent frame
	4. _top - Opens the document in the full body of the window
ex: <a href="https://www.w3schools.com/" target="_blank">Visit W3Schools!</a>

4. Relative URL: a link to a page within the same website
ex: <p><a href="images/do more.jpg">Work Hard</a></p>

5. Absolute URL: link another full website page
ex: <p><a href="https://www.w3.org/">W3C</a></p>

6. Mail: to create a link that opens the user's email program (to let them send a new email)
ex: <p><a href="mailto:avreetkaur084@gmail.com">Send Mail</a></p>

7. Button as link: 
ex- <p><button onclick = "document.location = 'doc/HTML_cheatsheet.txt'">HTML Cheatsheet</button></p>

8. Link titles: title attribute specifies extra information about an element.
ex: <a href="https://www.w3schools.com/html/" title="Go to W3Schools HTML section">Visit our HTML Tutorial</a>

#TABLE
1. <table>	 : Defines a table
2. <th>	 : Defines a header cell in a table
3. <tr>	 : Defines a row in a table
4. <td>	 : Defines a cell in a table
5. <caption> : Defines a table caption 
6. <colgroup>: Specifies a group of one or more columns in a table for formatting
7. <col>	 : Specifies column properties for each column within a <colgroup> element
8. <thead>	 : Groups the header content in a table
9. <tbody>	 : Groups the body content in a table
10.<tfoot>	 : Groups the footer content in a table

#BLOCK ELEMENTS - A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element.
EX: <p>, <div>
1. <div>: it is often used as a container for other HTML elements and is also used for a block of content. It has no required attributes, but style, class and id are common.

#INLINE ELMENTS - An inline element does not start on a new line.
EX: <span>
1. <span>: it is an inline container used to mark up a part of a text, or a part of a document. Style, class and id are common attributes for this element.
ex: <p>My mother has <span style="color:blue;font-weight:bold;">blue</span> eyes

#CSS kinds
1. Inline: using style element inside HTML elements
ex: <p style="color:red;">A red paragraph.</p>

2. Internal: using style element in the <head> section
ex: <head>
<style>
body {background-color: powderblue;}
h1   {color: blue;}
p    {color: red;}
</style>

3. External: using <link> element to link an external CSS file.
ex: <link rel="stylesheet" href="/html/styles.css">
